Multi-Window Invoice Reconciliation

‚úÖ THE AI SOLUTION
Window-by-Window Tracking: AI reconciles each 3.5-hour window separately with timestamp deconfliction for duplicate route codes.

Auto-Detection: Identifies 2 missing cancellations (routes CX12, CX15 never delivered) and 2 Flex pickups (routes AU11, AU12 with only 18 packages each, marked as rescues).

Auto-Dispute: Generates Chime message: "Request payment for Window 1 cancellations: CX12, CX15 (Nov 9). Request reclassify Window 2 routes AU11, AU12 from rescue to paid route."
